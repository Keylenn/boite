## 📦 安装
pnpm:

```sh
pnpm i @boites/react
```

yarn:

```sh
yarn add @boites/react
```

npm:

```sh
npm i @boites/react
```

## 🛠 本地调试

```sh
# 拉取项目
git clone https://github.com/Keylenn/boites.git

# 安装依赖
pnpm i

# 启动项目
pnpm -F react dev
```
